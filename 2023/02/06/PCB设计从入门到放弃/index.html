第一部分：课程介绍，学习方法及元器件库的创建

  1. 课程介绍以及学习思路介绍
    1. 学习目的：用AD做设计
    2. 学习内容：一是熟悉软件，二是原理图库以及原理图绘制，三是PCB布局以及PCB布线
    3. 怎么学：软件主要练习快捷键，设计主要是先看一遍，练习一遍，最后对比学习
  2. AD20的软件安装以及中英文切换
    1. 网上找了个蛮详细的，大家可以参考
    2. https://blog.csdn.net/weixin_40379143/article/details/108446461
  3. AD的工程组成以及创建
    1. 工程文件包含的内容：原理图库，原理图，PCB库，PCB，生产文件。 ==Tips：以上内容一定要包含在工程中，否则原理图与PCB无法交互。==
  4. 元件库的介绍及阻容模型的创建
    1. 元件符号是元件在原理图上的表现形式，主要由元件边框、管脚（包括管脚序号和管脚名称）、元件名称以及元件说明组成，通过放置的管脚来建立电器连接关系，元件符号中的管脚序号和电子元件实物的管脚一一对应的。在创建元件的时候，图形不一定和实物完全一样，但是对于管脚序号和名称，一定要严格按照数据手册中的的说明一一对应好。
  5. 在SCH Library中进行创建，主要参考元件的数据手册，注意电器属性引脚向外，可以修改引脚长度。
  6. IC类元件模型的创建
    1. 创建IC类元器件类型时，先放置外框，然后再放置引脚，最后再次修改名称即可。
    2. 如果元器件引脚上有横线，加个反斜杠（\）即可。
    3. 快捷键自定义：==放在对应的操作上，按住Ctrl，鼠标左键点击，即可设置新的快捷键。==
  7. 排针类元件模型的创建
    1. ==阵列粘贴==：类似于排针的引脚一样多个相同时候，可以使用本功能。先放置第一个引脚，==然后复制==，点击粘贴里面的阵列粘贴，输入对象数量，主增量，为管脚号，次增量，为管脚名称。点击确定即可快速添加多个引脚等。
  8. 光耦以及二极管元件模型的创建
    1. 创建例如三极管发射极和集电极，可以把栅格改小之后再去画。
  9. 现有元件模型的调用
    1. 以设计好的原理图为例可以生成原理图库，直接调用即可。

第二部分：智能小车的原理图绘制以及编译检查

  1. 元件的放置
    1. 创建好原理图库之后，直接安装此库，然后在右侧的components中直接添加即可
  2. 器件的复制以及对齐
    1. 放好的元器件直接复制：按住shift键，然后鼠标左键直接拉动即可。
    2. 放好的元器件直接移动：按住Ctrl键，然后鼠标左键直接拉动即可。
    3. 按空格键进行元器件方向的旋转。
    4. 按照模块化的思想进行原理图设计。
    5. ==Shift+Space 可以进行线的角度的设置。==
  3. 导线以及NetLable的添加
    1. 放置选项卡下的 线（Ctrl+W）是具有电气属性的线，而此选项卡最后绘图工具里面的线（L）是没有电气连接属性的。
  4. Value值的核对
    1. 统一标注：工具选项卡 -> 标注 -> 原理图标注，可以进行统一的标注。
  5. 封装的统一管理
    1. 工具选项卡 -> 封装管理器，里面可以针对同一类的元器件统一进行封装管理。
  6. 原理图的编译设置以及检查
    1. 需要报错的内容设置：工程选项卡 -> 工程设置、里面第一项可以设置不同错误的报错类型。然后在进行原理图编译的时候就可以看见报错。然后逐一核查即可。

第三部分：PCB封装库的创建方法及现有封装调用

  1. 常见CHIP封装的创建
    1. 注意创建PCB封装时，焊盘最外侧为阻焊层，目的在于防止绿油覆盖。
    2. 多个焊盘进行精确移动时，可以选中之后按M键，选择XY移动。
    3. 在复制状态下按住X或者Y可以进行器件的镜像。
  2. 常见IC类封装的创建
    1. 放置好第一个焊盘，之后的多个焊盘可以利用工具选项卡中的特殊粘贴，阵列粘贴进行统一的快速复制。
    2. 如果按照数据手册创建出现丝印覆盖焊盘的情况，可以利用编辑选项卡下的裁剪导线将导线进行切断处理。
  3. 利用IPC封装创建向导快速创建封装
    1. 首先需要安装此插件。未安装的在Extensions&Updates中进行安装。
    2. 制作IPC标准封装时，可以同时生成2D和3D效果图。
  4. 常用PCB封装的直接调用
    1. 用别人的或者自己的生成的库，进行安装即可直接调用。
  5. 3D模型的创建和导入
    1. 可以自己生成简易的3D体。不带引脚等。
    2. https://www.3dcontentcentral.cn/Default.aspx 也可以去此网站搜索3D模型，个人使用的话还是比较全的。
    3. 国内的IC原理图库和封装等。https://www.iclib.com/

第四部分：网表导入以及模块化布局设计

  1. 导入常见报错以及解决办法
    1. unknown pin
      1. 没有封装
      2. 封装管脚缺失
      3. 管脚号不匹配
    2. 常见绿色报错的消除
      1. 整体绿色是因为导入进去的红框没删掉。
      2. 如果两个元器件之间出现绿色报错是因为小于安全间距。
  2. PCB板框的评估及叠层设置
    1. 器件整齐摆放：工具选项卡下 -> 器件摆放，在矩形区域排列。即可在规定区域内放置所有元器件。
    2. 这部分的重要性：可以根据元器件摆放的矩形框大小，评估板材的尺寸。最后注意取整。
    3. 定位孔按照先放顶角，而后往内偏移的方式进行确定。
    4. 两层板与四层板区别：一是成本，二是设计难度，三是信号质量
    5. 层叠设置：设计选项卡 -> 层叠管理器，在里面添加所需要的层数即可。
    6. 注意正片层与正片层的区别：正片层只有画线的地方有铜皮，负片层只有画线的地方无铜皮。
  3. PCB快捷键的设置以及推荐

    1. 在需要设置快捷键的命令上按住Ctrl，重新设置快捷键。

      1. 快捷键推荐设置：

        拉线 过孔 覆铜 器件排列离散 线选 框选
        F2 F3 F4 F6 2 3
        左对齐 右对齐 上对齐 下对齐 水平等间距 垂直等间距
        Num4 Num3 Num8 Num2 Num7 Num9
        器件位号排列 差分线 删除网络 物理选择 执行DRC 规则
        Num5 Alt+F2 Num8 Ctrl+H TD DR
        Class 网络显示关闭 移动 选择 单位切换 单层显示
        DC N M S Q Shift+S
        切换抓取 多根走线 忽略障碍物
        Shift+E TTM Shift+R
  4. 模块化布局规划
    1. 通过工具选项卡下交叉选择模式可以将原理图选中的在PCB中高亮显示，使用此模式时，可以将操作区域垂直分布。
    2. ==交叉选择模式下，选中对应模块的元器件，然后在PCB界面中对应的进行器件排列，非常的好用，能够快速的将器件进行模块化分类。==
    3. 几大布局原则：
      1. 先大后小。
      2. 如果一个部分整体已经布局好，作为一个整体的话，右键，联合，从选中的器件生成联合。
      3. 交互式布线时，选中器件按快捷键 TC，可以定位器件。
      4. 信号流向原则。
      5. 考虑整体协调，布局美观。

第五部分：PCB规则设置以及PCB手工布线

  1. Class、设计参数、规则的创建

    1. Class的创建：设计选项卡 -> 类，即可添加相同的类。比如将所有的电源线统一加入Power类，即可进行统一隐藏或者显示。

    2. 设计参数设置：设计选项卡 -> 规则。可以对相对应的部分进行规则设置。此部分一共有九个大类。一是电气性能规则、二是信号线宽规则、三是贴片规则、四是阻焊规则、五是铺铜规则、六是测试点规则、七是生产部分规则、八是高速部分规则、九是放置器件的规则、最后是信号完整性分析。

      常用的有以下几类：

      1. 间距规则，与生产挂钩，具体与PCB板厂的生产能力相关。可查询对应厂家可支持的规则。通常为6mil以上。
      2. 线宽与最小孔径同理，参照厂家生产能力。线宽推荐6mil以上，最小孔径12mil以上。
      3. 例如此部分参照嘉立创工艺参数。|https://www.jlc.com/portal/vtechnology.html

    3. 电源部分线的载流宽度需要注意。电流大需要加粗。

    4. 过孔规则计算公式：2*H ± 2mil。

    5. 反焊层设置为全连接，推荐设置为8mil。

    6. 焊盘连接方式：载流情况下推荐使用全连接，因为要走大电流。但是在作为信号连接的情况下推荐使用十字连接。

    7. 过孔连接方式：建议采用全连接。

    8. 丝印与阻焊层的距离：推荐设置为2mil

  2. 扇孔的处理以及覆铜插件的应用
    1. PCB设计中先进行扇孔的目的：一是打孔占位、二是减少回流路径。
  3. PCB布线——信号线的走线
    1. 先考虑连通性
  4. PCB布线——电源部分的处理
    1. 考虑电源载流能力，注意线宽。
  5. 信号线的优化和GND的处理
    1. 如果出现尖端铜，可以进行多边形铺铜挖空。避免出现尖端放电效应。

第六部分：PCB的DRC检查，拼板设计以及资料输出

  1. DRC的检查以及丝印的调整
    1. 进行DRC检查的目的：前期为了满足各项设计的要求，通常会设置很多约束规则，当一个PCB设计完成之后，通常要进行DRC检查。DRC检查就是检查设计是否满足所设置的规则，一个完整的PCB设计必须经过各项电气规则的检查，常见的包括间距、开短路的检查。更加严格的还有差分对、阻抗线等检查。
    2. 问题：
      1. 如果出现警告弹窗，通常是因为修改铜皮之后没有进行重新铺铜。
      2. 如果出现啥都没有的情况，可能是因为检测时间不够，修改到5000ms以上。
      3. 丝印的DRC间距推荐设置为2mil。
    3. 丝印的调整
      1. 针对后期元器件装配，特别是手工装配元器件，一般都得出PCB的装配图，用于元件放料定位之用。
      2. 丝印位号不上阻焊，防止丝印生产之后缺失。
      3. 丝印位号清晰，字号推荐字宽/字高尺寸为4/25mil，5/30mil，6/45mil。
      4. 保持方向统一性，一般在一块PCB上不要超过两个方向摆放，推荐字母在左或在下。
  2. 什么是拼版？为什么要拼版？
    1. 节约生产成本
    2. 提高SMT生产效率
    3. 节省PCBA拿放时间
    4. 拼版的好处
      1. SMT产线的最大瓶颈，就是【锡膏印刷】制程，因为不论PCB的尺寸多大，其印刷所花费的时间都是25s左右，如果后期的快速贴片打件机、泛用贴片打件机所花费的时间如果小于锡膏印刷机，就会空等，造成资源的浪费。
      2. 其实贴片机的速度非常快，快到一秒钟可以打好几个零件，有些贴片机甚至有好几个吸嘴可以同时打件，以手机上PCBA上面的零件数来看，如果只有单一片板子，应该不到10s就可以完成所贴片作业。所以将PCB做拼版就可以增加贴片的零件数，也就可以增加贴片机的利用率。
  3. 拼板类型
    1. V-Cut
    2. 邮票孔：一般邮票孔使用孔径为直径0.8mm大小的非金属化孔，孔与孔的中心间距1.1mm左右，孔的个数5个为宜。
    3. 选用方式：规则的矩形使用V-Cut，不规则的、圆形等使用邮票孔
    4. 如果需要对类似于电源部分添加大面积铺地孔，则可以通过工具选项卡下，缝合孔进行整体添加。栅格150mil
  4. Gerber文件的输出以及整理
    1. 装配图的输出
    2. BOM文件的输出
      1. 报告选项卡下Bill of materials 可以进行输出
    3. Gerber文件的输出
      1. 文件选项卡 -> 装配输出 -> Gerber Fils，设置类型：单位选择英寸，格式选择2:4，绘制层可以全部选择，镜像层全部取消。
    4. 转孔文件的输出
    5. 坐标文件的输出
    6. 文件的整理
